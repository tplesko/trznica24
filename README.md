# Trznica24

Web aplikacija za simulaciju online tržnice. Koristi PHP za backend logiku i omogućava prikaz proizvoda, dodavanje u košaricu i osnovnu interakciju korisnika s ponudom.

## Tehnologije
- PHP
- HTML
- CSS
- JavaScript

## Opis funkcionalnosti
- Prikaz proizvoda
- Upravljanje košaricom
- Backend logika kroz PHP
